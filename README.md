## 我需要做一个待办事项的应用网页，要求：

#### 功能需求：

添加待办事件添加到待办中，待办实践可以分类，并根据分类统计频率，分类可以勾选系统提供的默认选项（例如学习，吃饭，寄快递等）也可以自定义输入。
每个任务的后面需要有标记框，可以勾选，勾选后代表完成，有流畅的动画移除待办列表，并计入频率统计。
每个任务后需要有删除按钮，和完成按钮一个在左侧一个在右侧，删除后移除该任务，并不计入统计。
显示已完成和未完成的任务数量
需要有事件的优先级设置，默认为低（其余为中、高、紧急）

#### 界面要求：

使用简单现代的设计风格，以温馨简洁为主调
色彩请使用白色、米白色、黄色等温馨暖色，优先级从低到高分别为淡绿色、蓝色、橙色、红色
圆角按钮和输入框，卡通风格
适当的阴影效果
响应式设计，适配手机
有少量适当的卡通装扮

#### 技术要求

前端技术选择React+Typescript+vite，状态管理用Zustand样式用Tailwind CSS,日期处理使用date-fns
数据保存在浏览器本地存储，刷新页面不会丢失
